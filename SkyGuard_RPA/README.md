# Sejam bem vindos ao SkyGuard Bot

###ELe é responsável por ler o site do SkyGuard e preencher um formulário.
###Este formulário tem a função de sinalizar quais são os ambientes com riscos críticos e altos, e anotar suas informações como localização, dias sem chuva, intensidade do foco de calor e bioma.

###Para rodar o projeto corretamente pegue o caminho da pasta(RPA - GS) onde está o index.html e cole na seção Take ScreenShot e cole no campo File name, para o print ir para o lugar correto
